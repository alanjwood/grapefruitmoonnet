Inspired by the <a href="http://minimalmac.com/enough" target="_blank">Enough</a> podcast and <a href="http://minimalmac.com/" target="_blank">Minimal Mac</a> site, both by <a href="https://twitter.com/#!/patrickrhone" target="_blank">Patrick Rhone</a>, I've been thinking about how little software I'd need to do all the things I need to, both for work and leisure purposes. &nbsp;I've been patiently waiting on the new Macbook Air refresh, as I'm looking to replace my 15&#8243; 2008 Macbook Pro. &nbsp;I've still not decided if I'm going 11&#8243; or 13&#8243;, although the latter is looking more likely. &nbsp;The base SSD size is 128GB for the current models. &nbsp;That doesn't seem very big, but I think I could easily get away with it, and have plenty to spare. &nbsp;In fact, I'm sure the 11&#8243; 64GB model would be fine too. &nbsp;So what would I install? &nbsp;Read on&#8230;

First up, without a doubt, would be <a href="http://www.google.com/chrome/" target="_blank">Google Chrome</a>, my browser of choice. &nbsp;I made the move to Chrome a couple of years ago from Firefox and it has went from strength to strength. &nbsp;Firefox was way too slow and clunky back then, and I really don't miss it at all. &nbsp;I've never liked Safari either, I did try it for a while, but found it's features too limited. &nbsp;Chrome is fast, extensible and fits perfectly into my workflow.

So that's the browser sorted, what about email? &nbsp;I'm a Gmail user, and the web client is excellent, but it's nice to have a dedicated desktop application. &nbsp;Sticking to a minimal &#8220;one thing well&#8221; theme, my client of choice is <a href="http://www.sparrowmailapp.com/" target="_blank">Sparrow</a>. &nbsp;I've been using this since the beta, and I love it, it gets better with every update. &nbsp;Mail.app, Thunderbird and Outlook are all too heavy and they don't play particularly nicely with Gmail for my liking.

<a href="http://agilebits.com/products/1Password/Mac" target="_blank">1Password</a> would be next. &nbsp;I use strong passwords generated and stored by 1Password, without it I'd never remember how to login to most sites, as each password is different. &nbsp;Really couldn't get anything done without this most excellent piece of software, well worth the investment, which at $39.99 is nothing compared to the added security it gives you.

Next up is a fairly new app for me, but one that has completely changed the way I use my Mac in the short time since I installed it. &nbsp;Step up <a href="http://www.alfredapp.com/" target="_blank">Alfred</a>! &nbsp;Kind of a hard app to describe, as it does so much, but it's a productivity app/launcher which has turned me into a serious keyboard ninja. &nbsp;With the paid-for Powerpack, it's an incredibly powerful and extensible piece of kit. &nbsp;An absolute must have. &nbsp;Already I'm finding it odd when I work on machines where it is not installed.

I'm a big Twitter user (I'm <a href="https://twitter.com/#!/grapefruitmoon" target="_blank">@grapefruitmoon</a>), but I'm not keen on their web interface. &nbsp;So <a href="http://itunes.apple.com/us/app/twitter/id409789998?mt=12" target="_blank">Twitter for Mac</a> would go on too. &nbsp;I've tried a few client apps, but I like the simplicity of this one. &nbsp;Others such as Tweetdeck are too busy for my liking, just too much going on.

<a href="http://www.dropbox.com/" target="_blank">Dropbox</a> is another must have. &nbsp;The ability to sync my documents across all machines and also be able to access them online is essential, and Dropbox does this perfectly. &nbsp;I don't use a great deal of the storage space available, but Selective Sync lets me choose exactly what folders I can sync to each machine, meaning I can save a bit of space if required.

For note-taking, my go-to app has been <a href="http://www.evernote.com/" target="_blank">Evernote</a>, but recently I've started experimenting with <a href="http://simplenoteapp.com/" target="_blank">Simplenote</a> and <a href="http://notational.net/" target="_blank">Notational Velocity</a>. &nbsp;I like the simple text-based format of the latter, very simple to use and importantly, very easy to get data in and out. &nbsp;I still like Evernote and use it for PDFs, links and image-based notes. &nbsp;But the issue with getting data out is one that bothers me somewhat, I'm really not a fan of proprietary formats. &nbsp;I'm not overly impressed with their iOS apps either, editing text based notes is not as simple as it should be. &nbsp;Simplenote on the other hand is perfect for this. &nbsp;Anyway, for the time being, I'd be installing both.

One of my most favourite Mac apps is <a href="http://www.literatureandlatte.com/scrivener.php" target="_blank">Scrivener</a>. &nbsp;For me, this is the best writing tool out there. &nbsp;It's not the kind of app I usually take to, given that it is extremely feature-heavy, but it makes writing, in it's many forms, an enjoyable experience. &nbsp;The ability to sync via Dropbox makes it a surefire winner.

OK, music and media next. &nbsp;With such a small storage capacity, I'm not looking to keep any music, photos or video locally. &nbsp;So Home Sharing via <a href="http://www.apple.com/itunes/" target="_blank">iTunes</a> is a given. &nbsp;When I'm not on my local wireless network, I'd use&nbsp;<a href="http://projectswithlove.com/streamtome/" target="_blank">StreamToMe</a> to get back to my media server. &nbsp;And I've recently subscribed to <a href="http://www.spotify.com/uk/" target="_blank">Spotify</a>&nbsp;Premium for streaming. &nbsp;I like Spotify, but I really want to love it; however, the glaring holes in the catalogue frustrate the hell out of me.

So that's pretty much it, that's essentially what I would need. &nbsp;For full on work purposes, I would really need <a href="http://www.vmware.com/products/fusion/overview.html" target="_blank">VMWare Fusion</a>, as I mainly work with <a href="http://www.microsoft.com/sqlserver/en/us/default.aspx" target="_blank">MS SQL Server</a>, but that's not reasonable for a laptop with little storage. &nbsp;I'd consider installing the various flavours of <a href="http://www.navicat.com/" target="_blank">Navicat</a> for connecting to remote SQL Server, mySQL and PostgreSQL instances.

Other apps I'd consider: <a href="http://www.hogbaysoftware.com/products/writeroom" target="_blank">Writeroom</a> (nice distraction-free writing app, although not an essential if I've got Scrivener), <a href="http://www.marinersoftware.com/products/macjournal/" target="_blank">MacJournal</a> (good journalling app, but could use Simplenote or Evernote for the same purpose) and <a href="http://www.skype.com/" target="_blank">Skype</a>. &nbsp;I don't particularly like Skype, but it's probably the most widely used VOIP and chat app available.

Compiling this list has made me realise just how little space and processing power I actually require these days. &nbsp;I'm conscious that I'm running a way over-powered Mac Pro at the moment, and I really don't need that much kit. &nbsp;When Lion is released, I'm going to experiment with using a Mac Mini as my main desktop. &nbsp;I've already gone from a two monitor set up to one (24&#8243;), and I'm not missing the second one at all. &nbsp;Streamlining and minimalism is the way to go!
